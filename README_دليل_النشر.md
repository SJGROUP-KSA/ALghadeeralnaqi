# موقع شركة الغدير النقي — دليل النشر

## محتويات الحزمة
- `index.html` — الموقع الكامل (صفحة واحدة، عربي/إنجليزي).
- `images/` — صور الكاتالوج (٨ صور محسّنة للويب).
- `Al_Ghadeer_Al_Naqi_Catalog.pdf` — الكاتالوج القابل للتحميل من الموقع.

الموقع يعمل بالكامل من جهة المتصفح، بدون أي خادم أو قاعدة بيانات.

---

## كيف ترفع الموقع على alghadeeralnaqi.com (الأسهل والأسرع)

### الخيار ١ — استضافة مجانية فورية (موصى به للبداية)
**Netlify** (مجاني):
1. ادخل على https://app.netlify.com وأنشئ حساباً.
2. اسحب مجلد الموقع كاملاً (`index.html` + `images` + الـ PDF) إلى صفحة "Add new site → Deploy manually".
3. خلال ثوانٍ يصبح الموقع مباشراً على رابط مؤقت.
4. من إعدادات الموقع → **Domain settings** → أضف `alghadeeralnaqi.com`، واتبع تعليمات تغيير الـ DNS عند مزوّد الدومين.

**Cloudflare Pages** بديل ممتاز بنفس الطريقة.

### الخيار ٢ — استضافة عادية (cPanel / Hostinger / مزوّد سعودي)
1. ادخل إلى لوحة الاستضافة → **File Manager**.
2. افتح مجلد `public_html`.
3. ارفع `index.html` ومجلد `images` وملف الـ PDF كما هي (نفس البنية).
4. الموقع يصبح مباشراً على دومينك فوراً.
